# Link Chip Hover Design

Clickable pill-shaped links should not look identical to informational pills.

## Decision

- keep `trust-chip` for informational, non-link pills
- add a dedicated interactive chip style for links
- interactive chips should show clear hover and focus feedback

## Interaction Direction

- stronger border on hover
- subtle accent-tinted background on hover
- accent text on hover
- visible focus ring

This preserves the existing visual language while making it obvious which pills are clickable.
