# What's New Grid Design

On the desktop What's New overview, release cards should not span the full width of the page. They should lay out side by side and expand to three columns when the viewport allows it.

## Decision

- keep a single column on mobile
- move to two columns on medium screens
- move to three columns on large desktop screens
- keep each card fully clickable as an anchor

This keeps the overview closer to an archive grid rather than a feed, and it matches the user’s request without changing the card content structure.
