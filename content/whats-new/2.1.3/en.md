## Personalization

Many of you wanted to customize the app a bit more to match your preferences. With this update, I can finally make that possible.

You can now not only change the app's accent color, but also choose the Intake app icon you like best. Here's a quick preview:

![Personalization](./assets/personalization.mp4)

## PDF export

This is especially useful for users who are working with a doctor or a nutrition coach and need an overview across several weeks that they can send or print.

There is now a new option in settings to create a PDF report for a selected time range. You can then share or save it using all the usual options. It includes all the important information and data you recorded in Intake during that period.

![PDF export](./assets/pdf-export.mp4)

## Nutrition overview

If you tap one of your macronutrient progress bars, a new overview opens that shows which foods contributed to it and in what amounts.

![Macro sheet](./assets/macro-sheet.mp4)

## Tracking time

You can now set a time for every product you log. By default, Intake uses the current time when you track something. This is important for PDF export so you can clearly see when you ate something.

## Bug fixes and improvements

As always, this release also includes a number of bug fixes based on the issues you keep reporting so diligently. Please keep it up :)
